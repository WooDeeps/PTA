# 7-1 输出等腰杨辉三角
输出6行等腰杨辉三角形。

### 输出格式:

第一行的1前面有13个空格；第2行的第一个1前面有11个空格，两个1之间有3个空格；最后一行的第一个1前面有3个空格。可以考虑每个数输出占4列列宽。

### 输出样例:

```out
           1
         1   1
       1   2   1
     1   3   3   1
   1   4   6   4   1
 1   5   10  10  5   1
```

```C
#include "stdio.h"

int main() {
    int a[6][6] = {0};
    int i, j;
    for (i = 0; i < 6; i++) {
        a[i][0] = 1;
    }
    for (i = 1; i < 6; i++) {
        for (j = 1; j < 6; j++) {
            a[i][j] = a[i - 1][j - 1] + a[i - 1][j];
        }
    }

    for (i = 0; i < 6; i++) {
        j = 10 - i * 2;
        for (; j > 0; j--) {
            printf(" ");
        }
        for (j = 0; j < i + 1; j++) {
            printf("%4d", a[i][j]);
        }
        printf("\n");
    }
    return 0;
}
```
